# Plan: unify await callee protocol to 1 return

## Goal

Collapse `f` and `__atmos` from `(ok, ret)` two-return to a single `r`,
matching the `until`/`while` predicate rule.

## Unified rule (callee returns one `r`)

- falsy      -> reject (keep waiting)
- `true`     -> accept, result = the event
- other      -> accept, result = `r`

Trade-off: cannot accept with a literal `false`/`nil` result (`0`/`""`
stay truthy in Lua, so they still work).
Behavior change: a plain-boolean `f`/`__atmos` now returns the event
(was `nil`).
